# STUDENT--RESULT--SYSTEM
A simple Python program to display student results using dictionaries, built for a PTM scenario...
A console-based Student Result Management System built in Python. Parents can enter their child's name to view subject-wise marks, simulating a Parent-Teacher Meeting (PTM) result display.
Description
This project simulates a simple PTM (Parent-Teacher Meeting) scenario where a parent enters their child's name, and the program displays the student's class, subjects, and marks using a Python dictionary.
Features
Displays a welcome message for the PTM session
Shows the list of registered students
Takes parent input (child's name) to fetch results
Displays class & section, subjects, and marks for each subject
Handles invalid names with a "record not found" message
Uses Python dictionaries for structured data storage
How it works
The program welcomes everyone to the PTM.
It displays the list of student names.
The parent enters their child's name.
The program searches the Results dictionary for a match.
